# 🏋️ Gym Membership Management System

A desktop application developed in **Java** using **Java Swing** to manage gym memberships through an interactive graphical user interface.

The project was developed as coursework for the **Programming** module during my BSc (Hons) Computing degree at **London Metropolitan University (Informatics College Pokhara).**

---

## 📖 Overview

The Gym Membership Management System is designed to simplify the management of gym members by allowing administrators to register, update, search and manage different membership plans.

The project demonstrates the practical application of Object-Oriented Programming (OOP) concepts while building a real desktop application using Java Swing.

---

## ✨ Features

- Register Regular Members
- Register Premium Members
- Store member information
- Update member details
- Display member records
- Manage different membership plans
- User-friendly graphical interface
- Input validation
- Object-oriented class hierarchy

---

## 👥 Membership Types
